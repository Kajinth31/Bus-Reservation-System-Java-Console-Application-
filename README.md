# Bus Reservation System (Java Console Application)

## Description
This is a **console-based Bus Reservation System** written in **Java**.  
It allows users to **register customers and buses, reserve and cancel seats, and manage a waiting list**. Data is persisted using text files, so the system retains information between sessions.

**Features:**
- Register new customers
- Register new buses
- Reserve seats on buses
- Cancel reservations
- Display all reservations
- Display available buses
- Manage a waiting list for full buses
- Persistent storage using `.txt` files
